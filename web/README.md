# DotagiftX UI

DotagiftX web user interface

### Tech Stack

- NodeJS 14
- React 17
- Next 10
- Material UI 4

### Features

- [x] market search
- [x] item market listing
- [x] popular items
- [x] newly listed items
- [x] user market listing
- [x] login with steam
- [x] user post item
- [x] user remove listing
- [x] user post set status and notes
